Work - Loop Animated Ad

An animation ad built with HTML, CSS, Javascript, and JQuery.

Designer and Programmer

April 2013

This was a fun task to create because it was the first time I really got to play with JQuery animate.

I was asked to make an ad using simple movement that was similar to PowerPoint movements. To achieve this movement, I chained jquery animates and stops while also tweaking the timing and css positions. The result was a simplistic ad template that could be used for convenience stores, clothing stores, or even mechanic shops.

The project was designed to fit a full HD 1920 x 1080 screen and sometimes it needed to be turned sideways to fit a sideways hanging screen.

I created the entire HTML, CSS, Javascript, JQuery, AJAX code.
